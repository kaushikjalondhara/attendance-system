# 📊 Attendance Management System (AMS)
A simple **Attendance Management System** built using **HTML, CSS, and JavaScript**.  
This project helps manage students and track daily attendance using browser localStorage.

---

## 🚀 Features

- 👨‍🎓 Add Students
- 📋 View Student List
- 📅 Mark Daily Attendance
- 📊 Dashboard with:
  - Total Students
  - Present Students
  - Absent Students
  - Attendance Percentage
- 🔄 Date-wise Attendance View
- 💾 Data stored in LocalStorage (No backend required)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
